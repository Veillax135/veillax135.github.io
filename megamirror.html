MegaMirror

MegaMirror is a smart mirror fork of MagicMirror², a modular, open-source, community-driven smart mirror Rasberry Pi application designed to run on a display behind a mirror, except Mega Mirror isn't meant to run on a 'Pi', instead, it is designed to run on any OS that will support a browser (including ChromeOS, as long as Visual Studio Code(VS Code) is installed).
