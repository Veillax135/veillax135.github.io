# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
|         | ❔                 |
| 2.0     | :white_check_mark: |
| 1.0     | :white_check_mark: |
| <1.0    | :x:                |

## Reporting a Vulnerability

Website will have a Report a Vulnerability page soon.

Tell them where to go, how often they can expect to get an update on a
reported vulnerability, what to expect if the vulnerability is accepted or
declined, etc.

## If a Vulnerability Report is Accepted❗
The linked attribute that has the vulnerability will be taen down untill further notice or untill vulnerability issue is resolved.
## If a Vulnerabillity Report is Denied❕
The vulnerability has either been resolved, is being resolved at the moment or is a faulty report.
